## Usage 

**1. Clone repositoy**
```bash 
git clone https://github.com/ch4n3-yoon/2022-KHU-SW-Mileage-Calculator.git
cd 2022-KHU-SW-Mileage-Calculator/
```

**2. Install required packages**
```bash
pip3 install -r requirements.txt
```

**3. Run!**
```bash
python3 main.py
```
